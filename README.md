# 🎯 سیستم AHP - پروژه حیات

## 🚀 راه‌اندازی فوری (رایگان)

1. روی دکمه سبز **Code** کلیک کنید
2. تب **Codespaces** را انتخاب کنید  
3. **Create codespace on main** را بزنید
4. صبر کنید 3 دقیقه...
5. بعد از آماده شدن، تب **PORTS** پایین را باز کنید
6. روی پورت **3000** کلیک راست → **Open in Browser**
